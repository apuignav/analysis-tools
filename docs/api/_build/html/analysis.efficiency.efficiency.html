efficiency¶

Efficiency class.

class analysis.efficiency.efficiency.Efficiency(var_list, config)[source]¶

Bases: object

Represent an efficiency object.

Initialize the internal configuration.

A rename-vars dictionary the config dictionary can be used to specify the expected name (in the data, in the ranges) of some of the variables in var_list.

Arguments:
var_list (list): List of observables to apply the efficiency to. config (dict): Efficiency model configuration.
