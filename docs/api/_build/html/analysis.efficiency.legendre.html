legendre¶

Efficiency expansion with Legendre polynomials.

class analysis.efficiency.legendre.LegendreEfficiency(var_list, config)[source]¶

Bases: analysis.efficiency.efficiency.Efficiency

Implement an efficiency with correlated Legendre functions.

Initialize the internal configuration.

In this case, the configuration of the efficiency model is as follows:
{‘pol-orders’:
{var_name1: n1,
var_name2: n2},

‘coefficients’: [coeff1, coeff2, …, coefn1xn2], ‘covariance’: [cov1, cov2, …] ‘ranges’: {var_name1: [min_var1, max_var1],

var_name2: [min_var2, max_var2]},

‘symmetric-variables’: [var_1]}

The range is used to rescale the data in the fit method. If it’s not given, it’s assumed it is [-1, 1].

Arguments:
var_list (list): List of observables to apply the efficiency to. config (dict): Efficiency model configuration.
Raise:
KeyError: On missing coefficients. ValueError: On bad range or bad symmetric variable defintion.

static fit(dataset, var_list, weight_var=None, legendre_orders=None, ranges=None, calculate_cov=False, chunk_size=1000)[source]¶

Calculate Legendre coefficients using the method of moments.

Arguments:

dataset (pandas.DataFrame): Data to model. var_list (list): Variables to model. Defines the order. weight_var (str, optional): Variable to use as weight. If None

is given, unity weights are used.

legendre_orders (dict): Variable name/max Legendre order. ranges (dict, optional): Ranges to scale the variables to. calculate_cov (bool, optional): Calculate the covariance matrix.

Defaults to False.
chunk_size (int, optional): Size of the chunks to calculate the
covariance matrix with. Defaults to 1000.
Return:
LegendreEfficiency: Multidimensional efficiency.
Raise:

ValueError: If the legendre orders are not given. KeyError: If some of the variables or the weight is missing from the

input dataset.

analysis.efficiency.legendre.scale_dataset(data, input_min, input_max, output_min, output_max)[source]¶

Rescale dataset to put it in the correct range.

Scaling does (input_min, input_max) -> (output_min, output_max).

The formula used to go from (input_min, input_max) to (output_min, output_max) is:

(output_max - output_min)(x - input_min)
f(x) = —————————————- + output_min
input_max - input_min
Arguments:
data (pandas.Series, numpy.array): Data to scale. input_min (float): Lower range of the input data. input_max (float): Upper range of the input data. output_min (float): Lower range of the output data. output_max (float): Upper range of the output data.
Return:
pandas.Series, numpy.array: Rescaled dataset.
