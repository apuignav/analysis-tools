analysis package¶

Generic analysis configuration.

analysis.get_global_var(name, default=None)[source]¶

Get a global variable.

Arguments:

name (str): Name of the variable. default (object, optional): Value to return if the global variable

is not defined. Defaults to None.
Return:
object: Value of the global variable.
analysis.set_global_var(name, value)[source]¶

Set value of a global variable.

Arguments:
name (str): Name of the variable. value (object): Value to assign to the global variable.
Return:
object: Value of the global variable.
