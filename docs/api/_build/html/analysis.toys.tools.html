tools¶

Common tools for managing toys.

analysis.toys.tools.load_toy_fits(*toy_list, **kwargs)[source]¶

Load toy fit results.

If several files are given, all the tables are merged.

Note:
The HDFStores are opened and closed. A copy of the data frames is returned.
Arguments:

*toy_list (list): List of toy names to load. **kwargs (dict): Extra options:

  • index (bool, optional): Index the data frame? Defaults

to True. + fail_on_incompatible (bool, optional): Fail when incompatible data frames are found? Defaults to True.

Return:
pandas.DataFrame: Merged data frame.
Raise:

OSError: If some of the toys do not exist. KeyError: If the data frames from the different stores are not compatible

and fail_on_incompatible is set.
